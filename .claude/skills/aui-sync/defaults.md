# AUI-Sync Defaults

## Paths

Configure your local paths here, or the agent will ask you.

- **Registry**: <path-to-assistant-ui>/apps/registry/components/assistant-ui
- **Templates**: <path-to-templates-container>
- **Examples**: <path-to-assistant-ui>/examples

## Instructions

If paths above contain `<placeholders>`, the agent will prompt you for:
1. Path to your local assistant-ui clone
2. Path to your templates container directory

To configure, replace the placeholders with your actual paths. For example:
- **Registry**: ~/code/assistant-ui/apps/registry/components/assistant-ui
- **Templates**: ~/code/aui-templates
- **Examples**: ~/code/assistant-ui/examples
