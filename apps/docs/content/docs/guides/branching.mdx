Branching lets users navigate between alternative versions of a message. A new branch is created when:

- A user message is edited
- An assistant message is reloaded (reload creates a new branch on the same message)

Branches are automatically tracked by assistant-ui by observing changes to the `messages` array.

## Shortest Working Pattern

<BranchingSample />

Place a branch picker inside your message component:

```tsx
import { BranchPickerPrimitive } from "@assistant-ui/react";

const BranchPicker = () => (
  <BranchPickerPrimitive.Root hideWhenSingleBranch>
    <BranchPickerPrimitive.Previous />
    <BranchPickerPrimitive.Number /> / <BranchPickerPrimitive.Count />
    <BranchPickerPrimitive.Next />
  </BranchPickerPrimitive.Root>
);
```

For the full primitive API and prop reference, see [BranchPickerPrimitive](/docs/primitives/branch-picker).

## API Hooks

These hooks require message scope and may only be called inside a message component.

### `useMessageBranching`

```tsx
import { useMessageBranching } from "@assistant-ui/react";

const { branchNumber, branchCount, goToPrev, goToNext } = useMessageBranching();
```

`goToPrev` and `goToNext` disable at branch boundaries. Both also disable while the thread is running unless the runtime supports `switchBranchDuringRun`. See [BranchPickerPrimitive](/docs/primitives/branch-picker) for that capability detail.

To navigate directly to a specific branch by id, use `useAui`:

```tsx
import { useAui } from "@assistant-ui/react";

const aui = useAui();
aui.message().switchToBranch({ branchId: "some-id" });
```

### `useActionBarReload`

```tsx
import { useActionBarReload } from "@assistant-ui/react";

const { reload, disabled } = useActionBarReload();
```

`disabled` is `true` when `thread.isRunning` or `thread.isDisabled`.

## Part Groups After Branching

Each branch is a distinct message version with its own content parts. `MessagePrimitive.PartGroups` enables hierarchical adjacent grouping of those parts, useful when a message mixes tool calls and text across branches. See the [MessagePrimitive](/docs/primitives/message) reference for usage.
