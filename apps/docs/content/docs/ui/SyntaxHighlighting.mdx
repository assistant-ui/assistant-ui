Syntax highlighting for code blocks in markdown.

<Callout type="warn">
Syntax highlighting is not enabled by default.
</Callout>

<Callout type="info">
`assistant-ui` offers two ways to add syntax highlighting to your markdown: 
- **react-shiki** (recommended for performance & dynamic language support)  
- **react-syntax-highlighter** (legacy - Prism or Highlight.js based)
</Callout>

---

## react‑shiki

<Steps>
  <Step>

#### Add the shiki‑highlighter component

```bash
npx shadcn@latest add "https://r.assistant-ui.com/shiki-highlighter"
```

Generates `/components/assistant-ui/shiki-highlighter.tsx` which uses [`react-shiki`](https://github.com/avgvstvs96/react-shiki) and installs the `react-shiki` dependency.

  </Step>
  <Step>

#### Add it to `defaultComponents` in `markdown-text.tsx`

```tsx title="/components/assistant-ui/markdown-text.tsx"
import { SyntaxHighlighter } from "./shiki-highlighter";

export const defaultComponents = memoizeMarkdownComponents({
  SyntaxHighlighter,
  h1: /* ... */,
  // ...other elements...
});
```

  </Step>
</Steps>

## react-syntax-highlighter (legacy)

<Steps>
  <Step>

#### Add the syntax-highlighter component

```bash
npx shadcn@latest add "https://r.assistant-ui.com/syntax-highlighter"
```

Generates `/components/assistant-ui/syntax-highlighter.tsx`, which uses `react-syntax-highlighter`.

  </Step>
  <Step>

#### Add it to `defaultComponents` in `markdown-text.tsx`

```tsx title="/components/assistant-ui/markdown-text.tsx"
import { SyntaxHighlighter } from "./syntax-highlighter";

export const defaultComponents = memoizeMarkdownComponents({
  SyntaxHighlighter,
  h1: /* ... */,
  // ...other elements...
});
```

  </Step>
</Steps>
