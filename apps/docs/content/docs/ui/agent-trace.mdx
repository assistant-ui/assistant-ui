### Pass a pre-shaped trace

`AgentTrace` does not inspect message parts. Shape the trace on your server or
runtime adapter, then pass it directly to the component.

```tsx
import {
  AgentTraceDisclosure,
  type AgentTraceNode,
} from "@/components/assistant-ui/agent-trace";

const trace: AgentTraceNode[] = [
  {
    kind: "step",
    id: "search",
    label: "Searched docs",
    type: "search",
    toolName: "search_docs",
    status: "complete",
    output: "Found the grouped message-part guidance.",
  },
  {
    kind: "step",
    id: "answer",
    label: "Drafted response",
    type: "complete",
    status: "complete",
  },
];

<AgentTraceDisclosure trace={trace} />;
```

</Steps>

## API Reference

### `AgentTraceDisclosure` Props

| Prop | Type | Default | Description |
| ---- | ---- | ------- | ----------- |
| `trace` | `AgentTraceNode[]` | - | Pre-shaped trace nodes to render. |
| `label` | `string` | `"Trace"` | Trigger label. |
| `summary` | `string \| (stats) => string` | - | Collapsed summary override. |
| `autoCollapseOnComplete` | `boolean` | `true` | Collapses after running trace steps complete. |
| `maxDepth` | `number` | `2` | Maximum nested group depth to expand. |
| `allowGroupExpand` | `boolean` | `true` | Enables nested group expansion. |

### `AgentTrace` Props

| Prop | Type | Default | Description |
| ---- | ---- | ------- | ----------- |
| `trace` | `AgentTraceNode[]` | - | Pre-shaped trace nodes to render. |
| `maxDepth` | `number` | `2` | Maximum nested group depth to expand. |
| `constrainHeight` | `boolean` | `true` | Caps timeline height and scrolls within the panel. |
| `allowGroupExpand` | `boolean` | `true` | Enables nested group expansion. |
| `nodeComponents` | `{ GroupSummary?, StepBody? }` | - | Replaces the default group summary or step body renderers. |
