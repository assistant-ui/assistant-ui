```tsx title="/components/assistant-ui/markdown-text.tsx"
import { MermaidDiagram } from "@/components/assistant-ui/mermaid-diagram"; // [!code ++]

const MarkdownTextImpl = () => {
  return (
    <MarkdownTextPrimitive
      remarkPlugins={[remarkGfm]}
      className="aui-md"
      components={defaultComponents}
      componentsByLanguage={{               // [!code ++]
        mermaid: {                          // [!code ++]
          SyntaxHighlighter: MermaidDiagram // [!code ++]
        },                                  // [!code ++]
      }}                                    // [!code ++]
    />
  );
};

export const MarkdownText = memo(MarkdownTextImpl);
```

## Configuration

Configure rendering options in `mermaid-diagram.tsx`:

```tsx title="/components/assistant-ui/mermaid-diagram.tsx"
renderMermaidSVG(code, {
  bg: "var(--background)",
  fg: "var(--foreground)",
  transparent: true,
});
```

The full palette is derived from your theme's background and foreground tokens, so diagrams follow light and dark mode automatically.

## Streaming Performance

The `MermaidDiagram` component is optimized for streaming scenarios:

- **Skeleton while streaming**: Shows a placeholder skeleton until the code block finishes streaming
- **Raw source fallback**: Invalid or unsupported diagrams fall back to displaying the raw source


## Supported Diagram Types

The component renders these diagram types:

- Flowcharts and decision trees
- Sequence diagrams
- Class diagrams
- State diagrams
- Entity relationship diagrams
- XY charts (bar, line, combined)

Other mermaid diagram types fall back to displaying the raw source. See the [Mermaid documentation](https://mermaid.js.org/) for syntax reference.
