## Overview

An open-source implementation of Claude Artifacts that allows users to generate interactive HTML content directly within the chat interface. The AI creates an artifact that users can open in a work panel alongside the conversation.

## Features

- **Chat + Work Panel**: Open generated artifacts beside the conversation
- **Source and Preview Modes**: Inspect the generated HTML or run it in the preview
- **Interactive Elements**: Artifacts can include scripts, forms, and animations
- **Iteration Support**: Ask the AI to update an existing artifact in place
- **Safe Execution**: Code runs in a sandboxed iframe with scripts enabled but no access to the parent application

## How It Works

1. **Request**: Ask the AI to create something (for example, "Create a calculator app")
2. **Generation**: The AI calls the artifact tool with a complete HTML document
3. **Inspection**: Open the artifact and switch between its source and sandboxed preview
4. **Iteration**: Request changes and the AI updates the existing artifact state

## Use Cases

- **Prototyping**: Quickly create UI mockups and interactive demos
- **Learning**: Generate code examples to understand concepts
- **Visualization**: Create charts, diagrams, and data visualizations
- **Games**: Build simple browser games and interactive experiences
- **Tools**: Generate calculators, converters, and utility applications
